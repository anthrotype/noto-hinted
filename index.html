Noto fonts hinted with ttfautohint

The noto-hinted repository contains various hinted versions for a bunch of Google's Noto fonts. Below are links to waterfall pages that demonstrate their rendering on HTML pages, together with PNG snapshot images that show how the HTML pages are rendered with Chrome, Firefox, and IE on MS Windows.

The snapshots were created automatically using BrowserStack, kindly providing an account to access its screenshot API.
